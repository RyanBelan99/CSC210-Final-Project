# CSC 210 Final Project Half Baked

Half Baked is a website using flask and Jinja

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Half Baked.

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
export FLASK_APP=app/app.py
```
To set to development run
```bash
export FLASK_ENV=development
```

## Usage

```bash
flask run
```
